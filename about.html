About This Project

The idea behind Legends XI.

What is Legends XI?

Legends XI is a football hall of fame and live scouting hub. It celebrates 15 players who permanently changed the sport through deeply researched biographical profiles, while the Live Scout page provides a live window into current football data via a third-party API. The project was built as an academic exercise using semantic HTML5, hand-written CSS3, Bootstrap 5, and vanilla ES6 JavaScript — no frameworks, no jQuery.

Technology Stack

The front-end is built with pure HTML5, hand-written CSS3 using CSS Custom Properties for a consistent design token system, Flexbox for all layouts, and Bootstrap 5 for the responsive grid. JavaScript uses modern ES6 features — classes, modules (import/export), async/await, and arrow functions — split into logical, single-responsibility files.

Curated Data

The Hall of Fame profiles are 100% original — each biography, legacy stat, and career summary was researched and written for this project. No AI-generated filler text was used for player bios. Statistical data (caps, goals, Ballon d'Or counts) is sourced from publicly available historical records.

Architecture Overview

The project follows a clean separation of concerns across four distinct layers: HTML (structure only — no inline styles or scripts), CSS (split into variables, base, components, and responsive files), Data (legends-data.js — pure export, no DOM logic), and JavaScript classes (each with a single responsibility: NavController, HallOfFameRenderer, ApiService, and ScoutRenderer).
